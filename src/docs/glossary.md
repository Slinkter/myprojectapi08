# Glosario Técnico

## Términos Generales
- **FBA (Feature-Based Architecture)**: Patrón de arquitectura donde el código se organiza por dominio de negocio (features) en lugar de por tipo de archivo.
- **SRP (Single Responsibility Principle)**: Principio SOLID que establece que un módulo debe tener una única razón para cambiar.
- **Dry (Don't Repeat Yourself)**: Principio de desarrollo de software para reducir la repetición de lógica.

## Librerías
- **React**: Librería de JS para construir interfaces de usuario.
- **Tailwind CSS**: Framework de CSS utilitario low-level.
- **Vite**: Build tool de próxima generación frontend.

## Conceptos del Proyecto
- **Alias (@)**: Referencia absoluta al directorio `src`.
- **Custom Hook**: Función de React que permite extraer lógica de componentes para ser reutilizada.
